# Get Started

	For do a new project we used:

```bash
poetry new name 
poetry init -> Just if we have a project, fill the params
```

## Add libraries

```bash
poetry add name
```

## Open Virtual Enviroment

```bash
poetry shell
```

## Create Txt Libraries

```bash
poetry export -f requirements.txt > requirements.txt
```

## Delete Libraries

```bash
poetry remove name
```
